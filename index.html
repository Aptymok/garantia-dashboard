Arquitectura real

GitHub Pages solo presenta interfaz. La ejecución segura vive en Microsoft 365.

  1. Dashboard en GitHub Pages dentro de SharePoint por iframe.
  2. Botón llama Power Automate con endpoint protegido.
  3. Flow ejecuta DAX contra Power BI / modelo semántico.
  4. Flow escribe resultado en Excel Online de SharePoint mediante Office Script.
  5. Copilot queda disponible como copiloto operativo del flujo.
